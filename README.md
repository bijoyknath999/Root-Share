# ⚡ RootShare P2P

**Direct device-to-device file transfer with zero server storage**

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![WebRTC](https://img.shields.io/badge/WebRTC-Enabled-green)](https://webrtc.org/)
[![Firebase](https://img.shields.io/badge/Firebase-RealTime-yellow)](https://firebase.google.com/)

## 🎯 Features

- ✨ **Modern Radar UI**: Visualize active rooms with a beautiful radar interface
- 🚀 **Direct Transfer**: Files go peer-to-peer, no middleman
- 📦 **Any File Size**: Send files of any size (20GB+ supported)
- 🔒 **Encrypted**: End-to-end encrypted via WebRTC
- ☁️ **No Storage**: Files are never stored on any server
- 📱 **Responsive Design**: Works perfectly on mobile and desktop
- 🌓 **Light/Dark Theme**: Choose your preferred theme

## 🚀 Quick Start

### Option 1: Use Online Version

Visit [RootShare P2P](https://your-username.github.io/Root-Share) (replace with your GitHub Pages URL)

### Option 2: Self-Host

1. **Clone or download** this repository
2. **Open `index.html`** in your browser
3. **Create a room** and share the code with your peer
4. **Select files** to send and start sharing!

## 🎮 How It Works

1. **Create a Room**: Click "Create Room" to generate a unique 6-character code
2. **Share the Code**: Send the code to your peer (or scan the QR code)
3. **Join the Room**: Your peer enters the code to connect
4. **Send Files**: Drag and drop files (or click to select) and watch them transfer!

## 🛠️ Tech Stack

- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **Real-time Database**: Firebase Realtime Database (for signaling)
- **Peer Connection**: WebRTC (for direct file transfer)
